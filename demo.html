🛠️ Spesifikasi Teknis

Detail implementasi dan fitur

💻 Teknologi

  • Backend: PHP Native (tanpa framework)
  • Database: MySQL dengan PDO
  • Frontend: HTML5, CSS3, JavaScript
  • Styling: Custom CSS (responsive)
  • Upload: Support gambar & video
  • PDF: Generate struk otomatis
  • WhatsApp: Integration via URL scheme

🚀 Fitur Unggulan

  • Role-based Access: Admin, Kasir, Marketing
  • Responsive Design: Mobile & desktop friendly
  • Real-time Updates: Stock & transaction sync
  • Security: Input validation & XSS protection
  • File Management: Upload dengan validasi
  • Reporting: Export CSV/Excel
  • cPanel Ready: Easy deployment

📋 Panduan Instalasi

Langkah-langkah setup di cPanel

✅ Langkah Instalasi

  1. Upload semua file ke public_html
  2. Buat database MySQL baru
  3. Import file daily_chicken.sql
  4. Edit config/db.php sesuai database
  5. Set permission folder uploads/ (755)
  6. Akses aplikasi via browser
  7. Login dengan admin/admin123
