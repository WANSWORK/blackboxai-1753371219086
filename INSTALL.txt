üìã QUICK INSTALLATION STEPS:

1. UPLOAD FILES
   - Extract daily_chicken_web_cashier.zip
   - Upload all files to your cPanel public_html directory
   - Maintain the folder structure

2. CREATE DATABASE
   - Login to cPanel > MySQL Databases
   - Create new database: daily_chicken
   - Create MySQL user with full privileges
   - Note down: database name, username, password

3. IMPORT DATABASE
   - Go to cPanel > phpMyAdmin
   - Select your database
   - Import > Choose file: daily_chicken.sql
   - Click "Go" to import

4. CONFIGURE DATABASE CONNECTION
   - Edit file: config/db.php
   - Update these lines:
     define('DB_HOST', 'localhost');
     define('DB_NAME', 'your_database_name');
     define('DB_USER', 'your_username');
     define('DB_PASS', 'your_password');

5. SET FOLDER PERMISSIONS
   - Set permissions 755 for these folders:
     uploads/
     uploads/logo/
     uploads/menu/
     uploads/promosi/
     pdf/

6. ACCESS YOUR APPLICATION
   - Open: http://yourdomain.com
   - Default login:
     Username: admin
     Password: admin123

7. FIRST TIME SETUP
   - Change admin password immediately
   - Upload your store logo (300x300px)
   - Create menu categories
   - Add menu items with prices
   - Create kasir and marketing users
